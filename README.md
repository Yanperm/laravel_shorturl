# Short Url
ดาวน์โหลด `git cloun`  
ติดตั้ง `composer install` และ `npm install`  
แก้ไขฐานข้อมูลที่ไฟล์ .env   
สร้างฐานข้อมูล พร้อมข้อมูล `php artisan migrate --seed`  
Run Laravel `php artisan serv` and Run `npm run watch`  
Enjoy :)
